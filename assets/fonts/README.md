# Premium Fonts for Kos Bae

## Inter Font Family

Download Inter font from: https://fonts.google.com/specimen/Inter

Required files:
- Inter-Regular.ttf (weight 400)
- Inter-Medium.ttf (weight 500)
- Inter-SemiBold.ttf (weight 600)
- Inter-Bold.ttf (weight 700)

### How to Install:

1. Visit https://fonts.google.com/specimen/Inter
2. Click "Download family"
3. Extract the ZIP file
4. Copy the following files from the `static` folder to `assets/fonts/`:
   - Inter-Regular.ttf
   - Inter-Medium.ttf
   - Inter-SemiBold.ttf
   - Inter-Bold.ttf

### Alternative: Use Flutter's bundled fonts

If you prefer to use system fonts, you can comment out the fonts section in pubspec.yaml and the app will fall back to the default font.

The Inter font provides a clean, modern, and premium look that's perfect for a luxury boarding house management app.
